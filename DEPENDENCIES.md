# Dependencies for the template

This template of a `discord.py` bot **REQUIRES** [`Python 3.10.x` or above](https://python.org/downloads).

The following requirements must be met before using this template:

- [`Python 3.10.x` or above](#installing-python)
- [Python must be installed on path](#install-python-on-path)
- [`pip`](#install-pip) must be installed
- [`venv`](#install-venv) must be installed
- Python packages from [`requirements.txt`](requirements.txt) must be installed

You can click the hyperlinks above to see how to install the clicked dependency or you can see how to install them below.
